## encode & decode
자바스크립트 인코딩, 디코딩

### 문자 인코딩
- btoa()  //  base64로 인코딩
- atob()  //  base64로 디코딩

### URL 인코딩
- encodeURI()
  : ; / = ? & 등을 제외하고 인코딩하는 함수
- encodeURIComponent()
  모든 문자를 인코딩하는 함수
- decodeURI()
  encodeURI() 로 인코딩한 문자열을 디코딩하는 함수
- decodeURIComponent()
  encodeURIComponent() 로 인코딩한 문자열을 디코딩하는 함수
